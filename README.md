# Cloud Industrie Stitch

Plateforme de gestion de magasin.

## Démarrage

1.  Installer les dépendances :
    ```bash
    npm install
    ```

2.  Configurer l'environnement :
    - Copier `.env.example` vers `.env` (si disponible) ou créer `.env`.
    - Ajouter `DATABASE_URL`, `NEXTAUTH_SECRET`, et les identifiants LinkedIn/Facebook.

3.  Lancer le serveur de développement :
    ```bash
    npm run dev
    ```

## Fonctionnalités

- Authentification via LinkedIn et Facebook.
- Intégration WhatsApp.
